## Security Terms
1. Threat = malicious event which exploits the vulnerability
2. Attack = assault on the system security caused by the threat
3. vulnerability = weakness in the security system which can be exploited
4. control = action, device, procedure or technique that removes or reduces a vulnerability

Security policies = highest level of documentation that describes the security procedures or protocols to be followed by the company.

## Types of Attack
There are two main types of attack

1. Passive Attack = attack which monitors the but does not intend to cause disruption/harm to the system
2. Active Attack = attack which collects the data and intend to cause disruption/harm to the system. Eg: Dos Attack
