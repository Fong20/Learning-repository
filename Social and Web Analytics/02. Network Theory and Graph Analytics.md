# 02. Network Theory and Graph Analytics

network consists of a group of nodes

## Components of a Network
1. Node = single entity within a network
2. Edge = connection between two nodes

## Types of Network
1. Directed graph = edge with a direction
2. Undirected graph = edge without a direction
3. weighted graph = edges with a strength value

## Centraility Metrics
Centrality measures identify which "nodes" (points) are the most important or influential.

There are 4 centraility metrics
1. Degree centrality
2. Betweenness centrality
3. Closeness centrality
4. Eigenvector centrality

## Network structures

Random network = all nodes have the same importance

small-world network = small network where all nodes have equal access to each other

scale-free network = few nodes have large connections while most nodes have only few connections
